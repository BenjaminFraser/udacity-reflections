How can you use the commands git log and git diff to view the history of files?
- You can use $git log to view all recent commits within the repository, via the git log. To scroll down and see further commits to the repository you can press the down arrow on the keyboard. To exit the log press q. You can use the command git diff first_id second_id to compare two commits, to see what lines and pieces of code have changed between the different commits. 

How might using version control make you more confident to make changes that could break something?
- Using git, you can confidently make changes, knowing that if something changes and no longer works, you can easily revert to older versions, or compare changes, to see quickly what it was that caused the issue to occur.
